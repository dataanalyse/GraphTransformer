# Repository Instructions

## Session Continuity
- At the start of each new task or fresh session, read `SESSION_NOTES.md` before doing substantive work.
- Use `SESSION_NOTES.md` as the primary handoff log for project context, decisions, open questions, and next steps.
- After meaningful progress, append a concise update to `SESSION_NOTES.md` so work survives IDE reloads or machine restarts.
- If `SESSION_NOTES.md` is missing, create it.
